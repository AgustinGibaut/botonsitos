Agustin Buttons

Descubre la manera más sencilla y rápida de agregar botones modernos a tu proyecto web:

npm install agustin-buttons
yarn add agustin-buttons
¡Listo para usar!

Prueba nuestros botones en distintos estilos, colores y efectos, todos listos para usar con clases simples.

Ejemplo de Uso

<!-- Agrega el CSS de Agustin Buttons --> <link rel="stylesheet" href="https://unpkg.com/agustin-buttons/dist/buttons.css"> <!-- Botones de ejemplo --> <button class="button-17">Botón 17</button> <button class="button-52">Botón 52</button> <button class="button-53">Botón 53</button>
